* 概念
堆其实是一种特殊的树，当树满足以下两点时，它就是一个堆:
- 完全二叉树
- 每个节点的值都必须大于等于或小于等于其左右子节点的值。
  + 每个节点的值都大于等于其左右子节点的值时，称为 =大顶堆= 
  + 每个节点的值都小于等于其左右子节点的值时，称为 =小顶堆=
