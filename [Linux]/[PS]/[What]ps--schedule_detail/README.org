* 调度器类（scheduler classes）
Linux 调度器以模块化的方式提供，这样允许进程可以有针对的选择调度算法。

Linux 会按照优先级顺序遍历调度器类，选择可执行进程的最高优先级调度器，然后去执行该调度器的任务。
