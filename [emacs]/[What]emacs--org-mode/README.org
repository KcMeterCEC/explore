** 编辑技巧

*** 跳转到专用的代码编辑模式
1. 在要写的代码块前后加上 `#BEGIN_SRC` `#END_SRC`，快捷输入方式 `<s - TAB`
2. 在代码块中使用使用快捷键 `C - c - '`，就会跳转到一个新 buffer 来编辑代码
3. 编辑完成后，使用 `C - c - '` 跳回 org-mode 模式
